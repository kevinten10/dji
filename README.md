## 🎮 立即体验

### 3D无人机模拟器
**浏览器中体验真实飞行！**

```
控制键：
W / S      → 上升 / 下降
A / D      → 左旋 / 右旋
↑ / ↓      → 前进 / 后退
← / →      → 左移 / 右移
R          → 重置位置
鼠标拖拽   → 旋转视角
滚轮       → 缩放视角
```

👉 打开 [simulation/drone-simulator/index.html](./simulation/drone-simulator/index.html)

---

### 无人机躲避游戏
**操控无人机躲避障碍物，看你能飞多远！**

```
控制键：
↑ ↓ ← → 或 WASD  → 控制飞行方向
```

👉 打开 [games/drone-game/index.html](./games/drone-game/index.html)
