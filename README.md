## 2. 飞行原理与结构

![无人机结构](https://images.unsplash.com/photo-1579829366248-204fe8413f31?w=800&h=400&fit=crop)

### 无人机主要结构

```
无人机结构图
┌─────────────────────────────────────┐
│              云台相机               │
│         (Gimbal + Camera)           │
├─────────────────────────────────────┤
│                                     │
│    ┌─────────────────────────┐      │
│    │        机身主体          │      │
│    │   (Flight Controller)   │      │
│    │      飞行控制器          │      │
│    └─────────────────────────┘      │
│                                     │
│      ↑         ↑         ↑         │
│     电机     电机       电机  电机   │
│    (Motor)  (Motor)   (Motor)(Motor)│
│                                     │
│      ↺         ↻         ↺         │
│     顺时针   逆时针     顺时针  逆时针 │
│                                     │
│    螺旋桨 (Propellers)              │
│                                     │
└─────────────────────────────────────┘
```

### 飞行控制原理

无人机通过调节四个电机的转速来实现各种飞行动作：

| 动作 | 原理 | 电机状态 |
|------|------|---------|
| **上升/下降** | 四个电机同时加速/减速 | 同时 +/- |
| **前后飞** | 前后电机差速 | 前后差异 |
| **左右飞** | 左右电机差速 | 左右差异 |
| **左旋/右旋** | 对角电机差速 | 对角差异 |

### 关键术语解释

| 术语 | 含义 |
|------|------|
| **姿态模式 (Attitude)** | 无人机自稳，但会有漂移 |
| **运动模式 (Sport)** | 解除限速，灵活飞行 |
| **三脚架模式 (Tripod)** | 缓慢飞行，方便构图 |
| **返航 (RTH)** | 一键返回起飞点 |
| **悬停 (Hover)** | 在固定位置保持静止 |

## 3. 法规与安全

![安全飞行](https://images.unsplash.com/photo-1508614999368-9260051292e5?w=800&h=400&fit=crop)

> ⚠️ **重要提醒**：2026年无人机监管已全面升级，违规飞行可能面临严重处罚！

### 中国无人机法规速查（2026年）

#### 必须完成的合规步骤

```
合规飞行 checklist
│
├── 📋 实名登记（必做）
│   ├── 250g以上必须登记
│   ├── 网址：uom.caac.gov.cn
│   └── 登记号需贴在机身
│
├── 🔖 运行识别（2025年12月起）
│   ├── 无人机需广播识别信息
│   └── 符合 GB 46750-2025 标准
│
├── 📜 飞行资质（看情况）
│   ├── 微型(<250g)：免证
│   ├── 轻型：免执照但需了解规则
│   └── 中型/大型：需CAAC执照
│
└── 🚫 禁飞区查询（每次飞前）
    └── 使用 UOM 平台查询
```

#### 飞行高度与区域限制

| 限制类型 | 具体要求 |
|---------|---------|
| **高度限制** | 真高 120 米以内 |
| **机场区域** | 绝对禁飞，需申请批准 |
| **军事禁区** | 绝对禁飞 |
| **人口密集区** | 需报备，建议不飞 |
| **铁路、电力设施** | 周边一定距离内禁飞 |

#### 违规处罚（2026年起）

| 违规行为 | 处罚 |
|---------|------|
| 未实名登记飞行 | 罚款 + 没收无人机 |
| 闯入禁飞区 | 罚款，严重者拘留 |
| 伤人/损物 | 民事赔偿 + 刑事责任 |

> 📖 **详细法规**：[安全飞行完全指南](docs/tutorials/safety-guide.md)

## 5. 基础飞行训练

![飞行训练](https://images.unsplash.com/photo-1507582020474-9a35b7d455d9?w=800&h=400&fit=crop)

### 四大基本飞行动作

#### 1. 上升与下降 (Throttle)

```
上升：缓慢向前推动左侧摇杆
      ┌─────┐
      │  ▲  │
      └─────┘

下降：缓慢向后拉回左侧摇杆
      ┌─────┐
      │  ▼  │
      └─────┘

技巧：缓慢平稳操作，避免急速升降
```

#### 2. 前后飞行 (Pitch)

```
前进：缓慢向前推动右侧摇杆
后退：缓慢向后拉回右侧摇杆
```

#### 3. 左右侧飞 (Roll)

```
左移：缓慢向左推动右侧摇杆
右移：缓慢向右推动右侧摇杆
```

#### 4. 旋转 (Yaw)

```
左旋：缓慢向左推动左侧摇杆
右旋：缓慢向右推动右侧摇杆
```

### 新手必练的 5 个基本功

```
飞行训练路径
│
├── 🅰️ 悬停练习
│   └── 在 2 米高度悬停 30 秒
│       目标：无人机不明显的漂移
│
├── 🅱️ 直线飞行
│   └── 飞出去 30 米再飞回来
│       目标：保持高度和方向稳定
│
├── 🅲️ 四边形航线
│   └── 围成一个 20 米正方形
│       目标：转角时保持平稳
│
├── 🅳️ 圆形航线
│   └── 绕一个中心点圆滑飞行
│       目标：速度均匀，不忽快忽慢
│
└── 🅴️ 8 字航线
    └── 飞一个躺着的 8 字
        目标：熟练掌握方向控制
```

### 常见错误与纠正

| 错误 | 原因 | 纠正方法 |
|------|------|---------|
| 起飞时机身倾斜 | 地面不平/螺旋桨问题 | 选择平坦地面起飞 |
| 飞行中漂移 | 指南针干扰/GPS不足 | 返航重新校准 |
| 无法起飞 | 视觉系统异常 | 检查传感器清洁度 |
| 遥控器失灵 | 信号干扰 | 远离干扰源 |

## 6. 进阶飞行技巧

![智能飞行](https://images.unsplash.com/photo-1487887235947-a955ef187fcc?w=800&h=400&fit=crop)

### 智能飞行模式

#### 📍 指点飞行 (Point of Interest)

设置一个兴趣点，无人机自动环绕拍摄：

```
使用方法：
1. 选中拍摄主体
2. 设置环绕半径和速度
3. 无人机自动完成环绕

适用场景：建筑、雕塑、大树等
```

#### 🎯 兴趣点跟随 (ActiveTrack)

让无人机自动跟随目标：

| 模式 | 适用场景 |
|------|---------|
| **跟随** | 跑步、骑行 |
| **平行跟随** | 车辆、船只 |
| **聚焦** | 人物转身面向镜头 |

#### 📐 航线飞行 (Waypoints)

预先规划航线，无人机自动飞行：

```
航线规划步骤：
1. 在地图上标记航点
2. 设置每个航点的高度和动作
3. 设置飞行速度和停留时间
4. 开始自动飞行

适用场景：大型场景延时、测绘
```

#### 🌅 延时摄影 (Hyperlapse)

自动生成稳定的延时视频：

| 模式 | 说明 |
|------|------|
| **自由模式** | 无人机自主飞行 |
| **环绕延时** | 环绕兴趣点拍摄 |
| **航线延时** | 沿预设航线拍摄 |
| **返航延时** | 自动返航中拍摄 |

### 特殊环境飞行技巧

#### 大风环境

```
大风飞行要点：
├── 选择有遮挡的区域起飞
├── 逆风飞行时加快速度
├── 顺风返航时控制好速度
├── 预留更多电量应对逆风
└── 风速过大时终止飞行
```

#### 低光环境

```
低光飞行要点：
├── 降低飞行速度
├── 开启补光灯辅助定位
├── 提高 ISO 或降低快门速度
├── 使用三脚架模式
└── 注意返航高度设置
```

#### 高原环境

```
高原飞行要点：
├── 气压低，飞行高度可能受限
├── 电池续航会明显下降
├── 选择无风或微风天气
└── 起飞前检查螺旋桨状态
```

## 7. 航拍参数设置

![航拍参数](https://images.unsplash.com/photo-1598550476439-6847785fcea6?w=800&h=400&fit=crop)

### 核心参数速查表

| 参数 | 推荐设置 | 说明 |
|------|---------|------|
| **格式** | RAW / D-Log | 保留更多后期空间 |
| **分辨率** | 4K 60fps | 流畅度高，便于后期 |
| **快门速度** | 1/50s - 1/120s | 根据光线调整 |
| **ISO** | 100-800 | 越低越好，控制噪点 |
| **白平衡** | 自动/手动 | 根据场景设置 |
| **对焦** | 手动对焦 | 确保主体清晰 |

### 快门速度参考

| 场景 | 建议快门 | 效果 |
|------|---------|------|
| 静止建筑 | 1/50s | 清晰无抖动 |
| 行走人物 | 1/100s | 主体清晰 |
| 行驶车辆 | 1/200s | 凝固瞬间 |
| 流水/瀑布 | 1/4s - 2s | 丝绢效果 |
| 星空 | 15s - 25s | 星轨效果 |

### 对焦技巧

```
对焦操作步骤：
│
├── 1. 点击屏幕上的主体
│   └── 确保对焦框对准拍摄对象
│
├── 2. 观察对焦提示
│   └── 绿色对焦框 = 对焦成功
│
├── 3. 锁定对焦
│   └── 长按锁定，避免跑焦
│
└── 4. 测光调整
    └── 滑动调整曝光补偿
```

### ND 滤镜使用指南

ND 滤镜可以减少进光量，让你使用更慢的快门速度：

| ND 型号 | 减少档位 | 适用场景 |
|---------|---------|---------|

```
航线设计要点
│

│   ├── 预留应急返航空间

│
├── 2. 画面丰富性
│   ├── 高度变化
│   ├── 角度变化
```

```
起点：低空近距离 → 逐渐拉远 + 上升
```

```
起点：主体前方 → 头顶飞越 → 主体后方
```

```
效果：逐渐展示全貌，气势递增
适用：古建筑群、岛屿、城堡
```

## 常见问题解答

![FAQ](https://images.unsplash.com/photo-1518005020951-eccb494ad742?w=800&h=400&fit=crop)

### 飞行问题

**Q: 无人机显示 "视觉系统故障" 怎么办？**
```
处理步骤：
1. 移到光线充足的环境
2. 清洁视觉传感器
3. 重新校准指南针
4. 重启无人机和遥控器
```

**Q: 飞行中断连怎么办？**
```
立即操作：
1. 保持冷静，不要乱动摇杆
2. 等待 3 秒，信号可能恢复
3. 若未恢复，无人机将自动返航
4. 观察无人机返航路径
5. 若能重新连接，可取消返航
```

**Q: 如何判断风速是否过大？**
```
判断标准：
├── 1-2 级风：适合新手
├── 3-4 级风：有经验飞手
├── 5 级风以上：建议不飞
└── 观察树叶、旗帜等参照物
```

### 拍摄问题

**Q: 为什么拍摄的画面有抖动？**
```
原因与解决：
├── 螺旋桨问题 → 更换螺旋桨
├── 电机故障 → 送修检查
├── 飞行太快 → 降低速度
└── 云台未校准 → 重新校准云台
```

**Q: 如何获得更清晰的画面？**
```
提升画质方法：
├── 降低 ISO（避免噪点）
├── 使用合适快门（1/50s 以上）
├── 对焦准确（手动对焦）
├── 使用 ND 滤镜（降低快门速度）
└── 拍摄时保持稳定悬停
```
