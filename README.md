# DJI 无人机项目

探索无人机的无限可能！

## 📁 项目结构

```
dji/
├── docs/                      # 文档目录
│   ├── tutorials/             # 教程
│   │   ├── beginner-guide.md  # 新手入门指南
│   │   └── safety-guide.md    # 安全飞行指南
│   ├── tips/                  # 飞行技巧
│   │   └── flight-tips.md     # 实用飞行技巧
│   └── ideas/                 # 创意灵感
│       └── fun-projects.md    # 有趣项目创意
├── simulation/                # 3D模拟器
│   └── drone-simulator/       # Three.js无人机模拟器
│       ├── index.html
│       ├── style.css
│       └── main.js
├── games/                     # 游戏项目
│   └── drone-game/            # 无人机躲避游戏
│       ├── index.html
│       ├── style.css
│       └── main.js
├── examples/                  # 代码示例
│   └── python/                # Python示例
│       └── drone_control.py   # 无人机控制示例
└── README.md
```

## 🚀 快速开始

### 3D无人机模拟器
直接在浏览器中打开 `simulation/drone-simulator/index.html` 即可体验！

### 无人机躲避游戏
打开 `games/drone-game/index.html` 开始游戏！

### 学习教程
查看 `docs/tutorials/` 和 `docs/tips/` 目录下的文档。
