# 实用飞行技巧

## 📹 航拍技巧

### 稳定拍摄
- 使用P模式或增稳模式
- 避免急转和快速加速
- 逆风飞行时降低速度

### 经典航拍镜头
1. **上升镜头**: 无人机垂直上升，视野逐渐扩大
2. **后退追踪**: 无人机向后飞行，镜头对准目标
3. **环绕盘旋**: 兴趣点环绕拍摄
4. **渐远镜头**: 无人机向后退同时上升
5. **穿越飞行**: 穿越狭窄空间（如窗户、桥洞）

### 光线运用
- **黄金时刻**: 日出日落前后，光线柔和
- **避免正午**: 阳光直射，阴影过重
- **顺光拍摄**: 色彩更鲜艳
- **逆光拍摄**: 可拍出剪影效果

## 🎮 飞行技巧

### 平稳起飞降落
1. 轻推油门，缓缓上升
2. 到达1米高度悬停确认
3. 降落时同样缓慢降低

### 精准悬停
- 关注无人机位置微调
- 利用GPS辅助保持位置
- 风大时适当调整姿态

### 环绕飞行
1. 先手动飞行一圈熟悉路线
2. 设置环绕点
3. 调整速度和高度
4. 保持匀速运动

## 🔧 设备优化

### 电池管理
- 准备多块电池
- 用温暖电池飞行（冬季）
- 避免电量耗尽再返航

### 图传优化
- 避开干扰源（WiFi、信号塔）
- 保持天线指向无人机
- 降低图传分辨率获得更远距离

## 💡 创意技巧

### 夜景拍摄
- 使用三脚架模式
- 降低ISO，增加曝光时间
- 使用ND滤镜

### 移动目标追踪
- 利用智能跟随功能
- 保持目标在画面中心
- 预判目标移动方向

### 低空飞行
- 开启下视避障
- 选择干净地面
- 减速慢飞

## 🛠️ 故障排除

### 常见问题
| 问题 | 解决方案 |
|------|---------|
| 飞行不稳 | 校准指南针 |
| 图传卡顿 | 重启设备，减少干扰 |
| 云台抖动 | 校准云台 |
| GPS丢失 | 移至开阔区域 |

### 固件更新
- 使用官方DJI Assistant 2
- 更新前备份设置
- 更新时保持电量充足
