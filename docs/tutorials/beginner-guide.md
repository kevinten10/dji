# 新手入门指南

## 了解无人机

### 无人机类型
- **消费级无人机**: DJI Mini、Mavic、Air系列，适合航拍爱好者
- **专业级无人机**: Mavic 3、Inspire，适合专业航拍和影视制作
- **FPV无人机**: 第一人称视角飞行，体验沉浸式飞行乐趣

### 关键术语
| 术语 | 解释 |
|------|------|
| 云台 | 相机悬挂装置，用于稳定画面 |
| 图传 | 实时传输画面到遥控器/手机 |
| 返航 | 无人机自动飞回起飞点 |
| 禁飞区 | 法律禁止飞行的区域 |

## 首次飞行准备

### 1. 检查设备
- ✅ 电池电量充足
- ✅ 螺旋桨安装正确
- ✅ 云台保护罩已移除
- ✅ 螺旋桨无损坏

### 2. 环境检查
- 选择开阔无人的场地
- 避开树木、建筑物、电线
- 确认天气适合飞行（避免大风、雨天）
- 远离机场、军事设施等禁飞区

### 3. 起飞前检查
1. 开启遥控器
2. 开启无人机
3. 等待GPS信号稳定（通常需要30秒）
4. 在DJI App中确认飞行条件
5. 校准指南针（如需要）

## 基本操作

### 遥控器布局（美国手）
```
左手操纵杆：
  ↑↓ 油门（上升/下降）
  ←→ 偏航（原地旋转）

右手操纵杆：
  ↑↓ 前后飞行
  ←→ 左右飞行
```

### 飞行模式
- **S模式（运动模式）**: 速度快，GPS辅助
- **P模式（定位模式）**: 默认模式，全向避障
- **N模式（普通模式）**: 限制速度，适合新手
- **F模式（增广模式）**: 支持APAS辅助绕行

## 维护与保养

### 电池保养
- 使用官方充电器
- 避免过充过放
- 长期存放保持50%电量
- 避免极端温度

### 机身清洁
- 定期清洁云台和电机
- 检查螺旋桨是否有裂纹
- 保持摄像头清洁
