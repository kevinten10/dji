# 安全飞行指南

## ⚠️ 绝对重要

飞行安全不仅是保护你的无人机，更是保护他人安全。2026年起，违规飞行将面临更严厉的处罚！

## 📋 核心法规（2026年最新）

### 1. 实名登记制度

根据 **GB 46761—2025**（2025年12月发布）：

| 无人机类型 | 实名要求 |
|-----------|---------|
| 250克以上 | **必须登记** |
| 250克以下 | 建议登记（自愿） |

**登记平台**：[UOM - 民用无人驾驶航空器综合管理平台](https://uom.caac.gov.cn)

**操作流程**：
1. 注册账号 → 2. 填写机身信息 → 3. 获取登记号 → 4. 粘贴于机身明显处

### 2. 操控资质要求

| 类别 | 重量 | 资质要求 |
|------|------|---------|
| 微型 | <250g | ✅ 免证（但需了解规则） |
| 轻型 | 250g~4kg | ✅ 无需执照，建议学习 |
| 小型 | 4kg~15kg | ⚠️ 需CAAC执照 |
| 中型 | 15kg~116kg | 🔴 需CAAC执照 |
| 大型 | >116kg | 🔴 需CAAC执照 |

**如何考证**：找AOPA认证的培训机构，理论+实操考核

### 3. 2026年新增要求

根据新修订的 **《治安管理处罚法》**（2026年1月1日施行）：
- 违规飞行将纳入治安管理处罚
- 部分敏感区域需要"飞手证"
- 处罚力度加大

### 4. 两项强制国标（2025年12月）

| 标准号 | 内容 |
|--------|------|
| GB 46761—2025 | 实名登记和激活要求 |
| GB 46750—2025 | 系统运行识别规范（无人机需广播身份信息） |

## 🚀 飞行规则

### 基本限制

```
高度限制：真高120米（AGL）
距离限制：保持在视距范围内（VLOS）
时间限制：白天飞行（除非有适当照明和许可）
区域限制：远离禁飞区
```

### 允许飞行区域

```
✅ 农村、农田
✅ 空旷公园
✅ 海滩（避开人群）
✅ 经申请批准的飞行区域
✅ 自己拥有的私有地产（需注意周围环境）
```

### 禁止飞行区域

```
❌ 机场净空保护区（通常5-20km）
❌ 军事禁区、军事管理区
❌ 核电站、水电站
❌ 铁路、电力设施周边
❌ 监管场所、看守所
❌ 车站、商场等人群密集区
❌ 禁止升空的区域
```

### 空域查询

使用 UOM 平台的空域查询功能：
👉 https://uom.caac.gov.cn

## ✅ 起飞前安全检查清单

```
□ 实名登记
  □ 已在UOM平台登记
  □ 登记号已粘贴于机身

□ 环境检查
  □ 天气良好（非雨天、大风>5级、雾霾）
  □ 非禁飞区（已通过UOM确认）
  □ 无电磁干扰
  □ 周围无人群

□ 设备检查
  □ 电池电量 > 80%
  □ 螺旋桨无损坏、安装牢固
  □ 电机和云台正常
  □ GPS信号良好

□ 软件设置
  □ 最新固件版本
  □ 返航高度设置正确
  □ 返航点已记录
  □ 已开启飞行限制
```

## 🌦️ 天气限制

| 天气 | 能否飞行 | 说明 |
|------|---------|------|
| 晴天 | ✅ | 最佳飞行条件 |
| 多云 | ✅ | 注意光线变化 |
| 小雨 | ❌ | 电子设备损坏风险 |
| 大风(>5级) | ❌ | 飞行不稳定 |
| 雾霾 | ❌ | 能见度低 |
| 低温(<0°C) | ⚠️ | 电池性能下降 |
| 雷暴 | ❌ | 危险！ |

## 🆘 紧急情况处理

### 失去控制信号
1. 等待信号恢复（通常几秒内）
2. 如未恢复，无人机将执行自动返航
3. 保持冷静，不要试图追着无人机跑

### 电量过低
1. 立即执行返航
2. 选择安全的降落点
3. 预留足够的电量储备（不少于20%）

### 电机异常
1. 立即停止飞行
2. 降落无人机
3. 检查并排除故障后再飞行

### 螺旋桨异物
1. 降落无人机
2. 关闭电源
3. 清除异物后再起飞

## 💰 保险建议

**强烈建议购买**：无人机第三方责任险
- 保额建议：至少50万
- 保费：每年几百元不等
- 用途：万一伤人或损物时可赔付
