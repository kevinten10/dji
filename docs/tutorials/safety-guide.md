# 安全飞行指南

## ⚠️ 绝对重要

飞行安全不仅是保护你的无人机，更是保护他人安全。

## 法规要求

### 中国大陆
- **实名登记**: 250g以上无人机需在民航局实名登记
- **飞行限制**: 120米以上高度需申请空域
- **禁飞区域**: 机场、军事区域、重要设施周边

### 通用规则
- 始终保持无人机在视距范围内 (VLOS)
- 不在人群上空飞行
- 不在夜间飞行（除非有适当照明和许可）
- 尊重他人隐私

## 起飞前安全检查清单

```
□ 环境检查
  □ 天气状况良好（非雨天、大风、雾霾）
  □ 无电磁干扰区域
  □ 非禁飞区

□ 设备检查
  □ 电池电量 > 80%
  □ 螺旋桨无损坏、安装牢固
  □ 电机和云台正常
  □ GPS信号良好

□ 软件检查
  □ 最新固件版本
  □ 返航高度设置正确
  □ 返航点已记录
```

## 紧急情况处理

### 失去控制信号
1. 等待信号恢复（通常几秒内）
2. 如未恢复，无人机将执行自动返航
3. 保持冷静，不要试图追着无人机跑

### 电量过低
1. 立即执行返航
2. 选择安全的降落点
3. 预留足够的电量储备

### 电机异常
1. 立即停止飞行
2. 降落无人机
3. 检查并排除故障后再飞行

### 螺旋桨异物
1. 降落无人机
2. 关闭电源
3. 清除异物后再起飞

## 天气限制

| 天气 | 能否飞行 | 说明 |
|------|---------|------|
| 晴天 | ✅ | 最佳飞行条件 |
| 多云 | ✅ | 注意光线变化 |
| 小雨 | ❌ | 电子设备损坏风险 |
| 大风(>5级) | ❌ | 飞行不稳定 |
| 雾霾 | ❌ | 能见度低 |
| 低温(<0°C) | ⚠️ | 电池性能下降 |

## 飞行区域建议

### ✅ 推荐区域
- 空旷的公园、草地
- 海滩（避开人群）
- 农村旷野

### ❌ 避免区域
- 机场周边 5km
- 军事设施区域
- 人口密集区
- 电力设施附近
- 高楼林立的城市中心

## 保险建议

考虑购买第三方责任险，以防万一。
